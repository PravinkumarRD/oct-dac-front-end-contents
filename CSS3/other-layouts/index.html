3️ - Column Layout

CSS3 column layout allows you to flow long text into multiple columns, just like in newspapers or magazines. It automatically balances content across columns and maintains consistent spacing. You can control the number of columns using column-count, spacing with column-gap, and even add a visual divider using column-rule. Resize your browser to see how the columns adapt to different widths. This is especially useful for displaying articles, lists, or descriptive text in a compact, readable form.
